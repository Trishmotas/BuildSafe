Featured Project

BuildSafe

BuildSafe is a BIM-based safety analysis prototype for Autodesk Revit supporting Prevention through Design (PtD) through rule-based and uncertainty-aware hazard identification workflows.

The project demonstrates how BIM-enabled automation can support design-stage safety awareness, structured hazard reporting, and transparent assumption-based decision-making.
